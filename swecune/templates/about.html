Data Source

Pokeapi.co

We scraped the Pokemon API by making requests for every one of our models, by iterating through each of the model names and concatenating the string to the end of a base URL. We can iteratively request information from the API which will be received in JSON form. Once we have all of the JSON files we can use python's json parser to parse the information into data structures which we can use in our templates.
